Frequently Asked Questions
##########################

**Can I install CartAGen using Conda?**

Not for now, you will need to install manually the package using pip inside your environment.

**I want to add an algorithm but I'm not a programmer**

You can open a new `issue <https://github.com/LostInZoom/cartagen/issues>`_ with the name
of the algorithm and some references (academic papers, already existing open-source code, *etc.*).
If the team thinks it would be a good addition to CartAGen, we'll try to implement it when we have time,
or maybe someone else will implement it.

Common issues
#############

To signal an issues with the CartAGen library or the QGIS plugin, you can open an issue
on their respective GitHub repository (`CartAGen library <https://github.com/LostInZoom/cartagen/issues>`_
and `QGIS plugin <https://github.com/LostInZoom/cartagen-qgis/issues>`_).
