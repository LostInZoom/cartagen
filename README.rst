========
CartAGen
========

`CartAGen <https://github.com/LostInZoom/cartagen4py>`_ is an open Python library dedicated to cartogaphic generalisation.
It is originally a port of the `CartAGen Java platform, <https://github.com/IGNF/CartAGen>`_ developed at IGN France and Univ. Gustave Eiffel.
